# Screenreader
## Steuerung:
### Shortcuts

- `H` Von Heading zu Heading
- `K` Von Link zu Link
- `Insert + F7` List all headings
- `1-6` Go to next heading of level [1-6]
- `D` Go to next landmark/region
- `T` Go to next table
- `L` Go to next list
